title=Craft Helper Continued
description=[h1]Craft Helper for build 41.65+[/h1]
description=
description=This is an updated version of [url=https://steamcommunity.com/sharedfiles/filedetails/?id=2186592938]Craft Helper (41.x) by b1n0m[/url] with additional features and improved UI.
description=
description=This mod will add a "Craft Helper 41" context menu option when right-clicking on any item with associated recipes.
description=Selecting that will bring up a UI with list of recipes that this item is part of, as well as ways to craft this item.
description=Selecting a recipe will show information about it (required items, skills etc). Result icon and required items are right-clickable.
description=Detailed information about interface can be found under ⓘ (info) symbol next to close button.
description=
description=Apart from item tab, there is also two new tabs:
description=[list]
description=[*] Search tab, where you can find all items and recipes (might be laggy with alot of mods)
description=[*] Favorite tab, where you can find all favorite [strike]items[/strike](WIP) and recipes
description=[/list]
description=
description=Safe to add/remove to/from existing saves
description=Should be compatible with all mods and MP
description=Does not support evolved recipes currently (soup, stew, salads etc.)
description=
description=[h2]Features:[/h2]
description=[list]
description=[*]Close window on ESC
description=[*]Category selector. Select desired category and filter that huge list of recipes.
description=[*]Window position saved to config. No more hassle with resizing/moving!
description=[*]Search bar. Supports custom rules (click on magnifier)
description=[*]Buttons to order list by name or type
description=[*]Ability to favorite recipes [strike]and items[/strike]. Even unknown ones!
description=[/list]
description=
description=[h2]Options ([url=https://steamcommunity.com/workshop/filedetails/?id=2169435993]Mod Options[/url] required to change):[/h2]
description=[list]
description=[*] Enable custom rules support (by default ON)
description=[*] Show icons of items in list (by default ON)
description=[*] Show hidden items/recipes (by default OFF)
description=[*] Close all tabs on close (by default OFF)
description=[*] Select font size for items in list (by default Large)
description=[*] Modifier keys (Shift, Ctrl) for some keybinds (by default all None)
description=[*] Show recipe origin (Mod) (by default ON)
description=[/list]
description=Also, with "Mod Options" you can set keybinds (Key bindings -> Craft Helper) for some actions.
description=
