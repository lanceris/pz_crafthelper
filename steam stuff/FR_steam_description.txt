[h1]Craft Helper pour Build 41.65+[/h1]

Ceci est une version mise à jour du mod [url=https://steamcommunity.com/sharedfiles/filedetails/?id=2186592938]Craft Helper (41.x) de b1n0m[/url] avec de nouvelles fonctionnalités et un IU amélioré.

Ce mod ajoute une option dans le menu contextuel clic-droit "Craft Helper" sur n'importe quel item associé à des recettes.
En faisant ça, cela affichera un IU avec la liste des recettes contenant cet item, mais aussi les différents moyens de crafter cet item.
Sélectionner une recette affichera les informations relatives à celle-ci (items requis, compétences etc). L'icône du résultat et les items requis sont cliquables avec clic-droit.
Les informations détaillées à propos de l'interface peuvent être affichées avec le symbole ⓘ (info) près du bouton pour fermer le menu.

Mis à part de l'onglet des items, il y a aussi deux autres onglets :
[list]
[*] Search tab, where you can find all items and recipes
[*] Favorite tab, where you can find all your favorite items and recipes
[/list]

Safe à ajouter/retirer d'une ou depuis une save existante
Devrait être compatible avec tous les mods et en MP

[hr][/hr]
[h2]Fonctionnalités :[/h2]
[list]
[*]Fermer un menu avec ESC
[*]Catégories. Sélectionnez une catégorie pour filtrer la géante liste de recettes.
[*]Les positions des menus est sauvegardée dans la configuration. Plus de problèmes liés avec le redimensionnement ou la position !
[*]Barre de recherche. Avec le support de règles customisées (cliquez sur magnifier)
[*]Boutons pour trier les listes par noms ou par types
[*]Ajouter des recettes ou des items aux favoris. Mêmes celles qui n'ont pas été apprises !
[*]Presets for favorite recipes and items.
[/list]

[hr][/hr]
[h2]Options ([url=https://steamcommunity.com/workshop/filedetails/?id=2169435993]Mod Options[/url] est requis pour les changer):[/h2]
[list]
[*] Enable custom rules for search (by default ON)
[*] Afficher les icônes des items dans la liste (ON)
[*] Afficher les items/recettes cachées (OFF)
[*] Fermer tous les menus à la fermeture (OFF)
[*] Select font size for items in list (Medium)
[*] Modifier les touches (Shift, Ctrl) de raccourcis (tous None)
[*] Affichier l'origine de la recette (Mod) (ON)
[*] Show favorite items in inventories (OFF)
[*] Enable editable category selector (OFF)
[*] Delayed search (Need to press Enter to perform search) (OFF)
[*] Behaviour when right-clicking items in inventory (only "Craft Helper" option)
[*] Window opacity (80%)
[*] Scroll speed (100 ms)
[/list]
Aussi, avec le mod "Mod Options" vous pouvez choisir vos touches (Key bindings -> Craft Helper) pour certaines actions.
